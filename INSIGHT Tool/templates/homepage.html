Get Started

1. Clone & Configure

Clone the repository and set up your environment variables (including Gemini API Key).

git clone https://github.com/your-repo/insight-tool.git
cd insight-tool/INSIGHT\ Tool
# Create .env file with your credentials

2. Install Dependencies

Install Python requirements and set up Neo4j.

pip install -r requirements.txt

3. Run & Connect

Start the server and expose it via Ngrok.

python main.py
ngrok http 5000
